# Data downloads

## The landscape spreadsheet

[:material-microsoft-excel: Download the spreadsheet](assets/BoyceLab_Genetic_Epilepsy_Landscape.xlsx){ .md-button .md-button--primary }

Four sheets:

**Master List** — All 1,078 genes from Genes4Epilepsy v2026-03 as a sortable Excel Table. Includes HGNC ID, Entrez ID, Ensembl ID, OMIM gene ID, inheritance, phenotype group, plus six external reference URLs (HGNC, NCBI Gene, ClinVar, gnomAD, GeneReviews search, Orphanet gene record) for every gene. Tier 1 and Tier 2 rows additionally have foundation, registry, platform, research center, therapeutic program, philanthropy, and social media data.

**Tier 1 Detail** — The 128 deep-dived genes filtered into a focused view for review.

**Methodology** — Tiering definitions, field conventions, caveats, and source list.

**Source Index** — Every URL used during verification, cataloged with what each source contributed.

## Source data

The underlying gene list comes from Genes4Epilepsy:

- **[Genes4Epilepsy v2026-03 release](https://github.com/bahlolab/Genes4Epilepsy/releases)** — TSV download, GitHub release page
- **[Genes4Epilepsy paper](https://onlinelibrary.wiley.com/doi/10.1111/epi.17547)** — Oliver KL et al. *Epilepsia* 2023

Genes4Epilepsy is updated twice yearly. When a new release is published, the landscape source list will be refreshed and reference URLs regenerated.

## Citing this resource

If you use this landscape in a publication, presentation, or other formal context, please cite both the underlying gene list and this resource:

!!! quote "Suggested citation"
    Boyce Lab Genetic Epilepsy Landscape. Compiled from Genes4Epilepsy v2026-03 (Oliver et al., *Epilepsia* 2023). Available at [boycedatascience.com/landscape](https://boycedatascience.com/landscape/). Accessed [date].

If you cite a specific deep-dive entry, **please verify the URLs and program details before citation** — therapeutic programs, foundation web presence, and registry platforms change frequently in rare disease.

## Reusing the data

The landscape spreadsheet is shared so the rare epilepsy community can reuse and remix it. We ask that you:

- Cite the source as above
- [Tell us about corrections or additions](about.md) so improvements flow back to the community
- Don't redistribute as your own work

If you're considering deeper integration (e.g., feeding this into a clinical decision-support tool or a foundation portal), please [reach out](about.md) — we'd love to hear what you're building.
