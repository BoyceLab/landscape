# Dashboard

The dashboard below covers all **1,078 genes** from the Genes4Epilepsy v2026-03 curated list, with deep-dive landscape data for the 128 genes that have established or emerging patient communities, and reference links for every gene.

!!! tip "Quick start"
    - Type a gene name, foundation, disorder, or platform in the **search box**
    - Use the **status filter** to narrow to Established, Emerging, or Background
    - Use **quick filters** (Has registry / Has foundation / Active therapeutic) to find genes with infrastructure
    - Click any **column header** to sort
    - **Platform pills** (Combined Brain, Sanford CoRDS, Simons Searchlight, etc.) link to the platform's home page
    - **Resource pills** (PubMed, HGNC, NCBI, ClinVar, gnomAD, GR, Orphanet) link to authoritative gene-level references
    - The **+ Suggest a correction** button (top right of the dashboard) opens a form that emails the maintainer

## Working with the dashboard

The dashboard is a single self-contained HTML widget — it loads no external trackers, stores no data, and works offline once loaded. If you'd prefer to work with the underlying data directly, see the [Data downloads](data.md) page for the spreadsheet.

## What's not in the dashboard

A handful of editorial notes — context about why a foundation is interesting, comparison to peer genes, and similar judgment calls — are kept separate from the public dashboard. These notes inform what's surfaced but aren't displayed publicly. If you'd like context on a specific entry, [reach out](about.md).
